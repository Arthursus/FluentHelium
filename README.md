# Fluent Helium 
Microlibrary to design and develop modular applications
## Target Platform
NetStandard 1.1 (.NET Framework 4.5 for unit tests only) 
## Dependencies
### Mandatory
* Immutable Collections
* Reactive Extensions
### Integration
* Autofac ()
### Unit testing
* xUnit
* Fluent Assertions
* NSubstitute
